
# git lfs
https://git-lfs.com/

For Gitea: https://docs.gitea.com/

## One-time setup per repository

```
git lfs install
```

```
git lfs track "*.png"
git lfs track "*.jpg"
```

For Termux, it's not pre-installed:
```
pkg install git-lfs
```

## Each client needs to run:

```
git lfs install
```
